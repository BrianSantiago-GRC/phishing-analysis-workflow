# Evidence Notes

This folder is for sanitized evidence notes connected to the phishing analysis workflow.

## Current Status

Workflow documentation is included. Evidence files should be added only when original screenshots or sanitized examples are available.

## Acceptable Evidence

- Sanitized email screenshot.
- Sanitized email header screenshot.
- Sanitized URL reputation screenshot.
- Completed IOC checklist.
- Completed incident notes.

## Do Not Include

- Real user information.
- Real customer or employer data.
- Private URLs.
- Message IDs.
- Tenant details.
- Credentials, cookies, or tokens.
