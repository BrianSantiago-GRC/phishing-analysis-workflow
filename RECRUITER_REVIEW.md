## What A Recruiter Should Notice

This project shows a structured phishing triage workflow. It is relevant for teams that need someone who can handle user-reported suspicious emails, preserve evidence, review indicators, classify risk, recommend containment, and document the decision path clearly.

## Proof Points

- Built a user-reported phishing intake and investigation workflow.
- Documented header, link, attachment, and IOC review concepts.
- Created incident notes and escalation summaries.
- Connected Microsoft 365 context to security operations documentation.
- Kept the work sanitized and junior-scope.
